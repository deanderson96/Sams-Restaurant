# Sam's Restaurant Website

Static website files for the `Sams-Restaurant` GitHub repository.

## Files

All files are kept in the same folder, with no subfolders:

- `index.html`
- `styles.css`
- `script.js`
- `README.md`

The menus are typed into the HTML as text. There are no image files or image references.

## Deploy on GitHub Pages

1. Upload these files to the root of the `Sams-Restaurant` repository.
2. Go to the repository on GitHub.
3. Open **Settings**.
4. Open **Pages**.
5. Set the source to **Deploy from a branch**.
6. Choose the `main` branch and `/root` folder.
7. Save.

GitHub will publish the site after a short build.
